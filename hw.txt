Сложная дз:

"Распакуйте Zip-файл, который вам отправили, и откройте HTML-файлы, которые находятся внутри. Добавьте их в вашу папку "templates", а папки (css, js, images, fonts) поместите в вашу папку "static". 

Затем внутри приложения settings создайте класс Settings. Внутри класса создайте следующие поля:

- `title`: поле с ограничением по символам для названия сайта (используйте также `verbose_name`).
- `logo`: поле для фотографии, в данном случае, для логотипа (используйте также `verbose_name`).
- `description`: поле без ограничений по символам для описания сайта (используйте также `verbose_name`).
- `phone`: поле с ограничением по символам для телефонного номера (используйте также `verbose_name`).
- `email`: поле для почты (используйте также `verbose_name`). Подсказка: используйте `EmailField()`.

После создания класса, зарегистрируйте его в административной панели. Также создайте функцию `index` для главной страницы и подключите к ней шаблон `homepage.html`. Подключите параметры шаблона из папки "static" к этой странице."
